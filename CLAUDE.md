# Oracle Template — Run /awaken to Create Your Identity

> This is a blank template. Run `/awaken` in Claude Code to generate your Oracle's soul.

## Quick Start

1. Open Claude Code in this directory: `claude`
2. Type: `/awaken`
3. Follow the wizard — name your Oracle, define its purpose
4. Your personalized CLAUDE.md will replace this file

## What /awaken Will Ask

- **Oracle name** — What should your AI agent be called?
- **Human name** — What should it call you?
- **Theme** — A motto or guiding principle
- **Purpose** — What will your Oracle do?
- **Language** — What language do you primarily use?
