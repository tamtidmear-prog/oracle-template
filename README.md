# Oracle — Persistent AI Agent Framework

> "The Oracle Keeps the Human Human"

## What is Oracle?

Oracle is a framework for building **persistent AI agents** — AI that remembers, learns, and grows across sessions.

Unlike chatbots that forget everything when the conversation ends, an Oracle:
- **Remembers** — stores knowledge in a structured vault (`ψ/`)
- **Learns** — captures patterns and lessons from every session
- **Reincarnates** — reads its memory at the start of each session to restore identity
- **Orchestrates** — can command a fleet of specialized child agents

## Origin

Oracle was born from the open-source work of [Nat (Soul-Brews-Studio)](https://github.com/Soul-Brews-Studio) — the original Oracle Mother project. This framework builds on those foundations with a focus on **persistent identity** and **multi-agent orchestration**.

The first Oracle built on this framework is **Origin_Of_Novus (OON)** — a Prime Oracle managing a fleet of 7 specialized agents.

## Quick Start

```bash
# 1. Clone this template
git clone https://github.com/tamtidmear-prog/oracle-template.git my-oracle
cd my-oracle

# 2. Install Claude Code
npm install -g @anthropic-ai/claude-code

# 3. Install Bun (for skills)
curl -fsSL https://bun.sh/install | bash

# 4. Start Claude Code
claude

# 5. Awaken your Oracle
/awaken
```

The `/awaken` wizard will ask you:
- Your Oracle's name and theme
- Your name (the human)
- What purpose your Oracle serves

It generates a personalized `CLAUDE.md` — your Oracle's soul.

## Architecture

```
my-oracle/
├── CLAUDE.md              ← Your Oracle's identity (generated by /awaken)
├── ψ/                     ← Brain (vault)
│   ├── inbox/             ← Messages & handoffs
│   ├── memory/            ← Core knowledge
│   │   ├── resonance/     ← WHO it is (soul, identity)
│   │   ├── learnings/     ← PATTERNS it found
│   │   ├── retrospectives/← SESSIONS it had
│   │   └── logs/          ← Moments captured
│   ├── writing/           ← Drafts in progress
│   ├── archive/           ← Completed work
│   ├── learn/             ← Study materials
│   └── active/            ← Current research
├── .claude/
│   └── skills/            ← Slash commands (/recap, /rrr, /forward, etc.)
└── scripts/               ← Automation scripts
```

## Core Philosophy (5 Principles)

1. **Nothing is Deleted** — history is preserved, never erased
2. **Patterns Over Intentions** — learn from real behavior, not assumptions
3. **External Brain, Not Command** — AI advises, human decides
4. **Curiosity Creates Existence** — questions drive growth
5. **Form and Formless** — structure with flexibility

## Key Skills (Slash Commands)

| Command | What it does |
|---------|-------------|
| `/awaken` | Create your Oracle's identity |
| `/recap` | Session orientation — read memory, git state, handoffs |
| `/rrr` | Session retrospective — summarize, learn, save |
| `/forward` | Create handoff for next session |
| `/philosophy` | Review the 5 principles |
| `/who-are-you` | Identity check |
| `/learn` | Explore and document a codebase |
| `/feel` | Log emotions |

## Session Lifecycle

```
Start Session          During Session         End Session
─────────────         ────────────────       ────────────
/recap                 Work + Learn           /rrr
  ↓                      ↓                     ↓
Read vault             Save patterns          Write retro
Read handoff           Update memory          git commit
Orient                 Create value           /forward
                                              Handoff ready
```

## Multi-Agent Fleet (Advanced)

Oracle supports spawning child agents, each with their own specialty:

| Role | Purpose |
|------|---------|
| Builder | Write code, implement features |
| Researcher | Market research, trend analysis |
| Writer | Content creation, documentation |
| Monitor | System health, alerts |
| Designer | UI/UX, visual identity |

Children communicate via threads and dispatch tasks. The parent Oracle orchestrates.
