# ψ/ — Oracle Brain (Vault)

This is your Oracle's brain. Knowledge flows through it:

```
active/ → memory/logs → memory/retrospectives → memory/learnings → memory/resonance
(research → snapshot → session → patterns → soul)
```

## Structure

| Directory | Purpose |
|-----------|---------|
| `inbox/` | Messages, handoffs, tasks |
| `memory/resonance/` | WHO your Oracle is (soul, identity) |
| `memory/learnings/` | PATTERNS it discovered |
| `memory/retrospectives/` | SESSION journals |
| `memory/logs/` | Ephemeral moments |
| `writing/` | Drafts in progress |
| `archive/` | Completed work |
| `active/` | Current research |
| `learn/` | Study materials |
| `outbox/` | Outgoing messages |
| `lab/` | Experiments |

## Privacy

All content in `ψ/` is **gitignored by default** (except this README and .gitkeep files). Your Oracle's memories stay on your machine.
